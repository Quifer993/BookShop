# BookShop

##Stack
- JDK – Java 1.8
- IoC - SpringBoot 2.x
- Front – Spring Web
- DB – PostgreSQL 14
- Сборщик: Maven

#Запуск
Для настройки бд Postgres запустите в корневой папке проекте 
```
docker-compose up
```
Для компиляции программы используется 
```
mvn package
```
После чего запускаем jar файл в папке target.

#Вход на страницу
Перейдите на localhost:8080 ([Локалка](http://localhost:8080/))
